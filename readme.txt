Replace Profile Account section inside Membership Pro 2 with a Formidable Pro Form

== Description ==
---
If you need a custom form from Formidable Pro to replace the User Account section inside of Membership 2, this is your solution.

### Follow this steps to create the solution:
 * You need to setup Membership 2, Formidable Pro and Formidable Registration Addon.
 * Create your Formidable form to register the users.
 * Create a page to hold that form.
 * Go to Membership 2 and setup the Formidable page as the registration page.
 * Then go to the MP2 Account page and change the shortcode to hide the User section.
 * Now go to this plugin and select the form and where you want to appear the form.
---

== Installation ==
Upload the entire gfirem folder to the /wp-content/plugins/ directory or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' menu in WordPress.
Head to the 'GFireM FFP-MP2' menu item in your admin sidebar to adjust the settings
